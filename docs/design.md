# UI Design

## Zoom Levels (yazi風)

3段階のズームで時間軸をシームレスに切り替え:

```
月ビュー ←h── 週ビュー ←h── 日ビュー
         ──l→          ──l→
```

- **日**: 15分ブロック単位のタスク一覧 + タイムラインバー
- **週**: 7日間の進捗・残り時間サマリー（lで日に潜る）
- **月**: カレンダー俯瞰

## Layout

2カラム構成:

- **左サイドバー**: 縦タイムライン（時刻 + ブロック記号）
- **右メイン**: タスクテーブル（カーソル操作の主領域）
- `t`キーで右メインをGoogleカレンダー風縦タイムライン（色付きブロック）に切替

> モック: `mock8.py`（Textual製プロトタイプ。技術選定とは独立）

## Competitive Landscape

### TUI/CLI
| Name | Diff from ytasky |
|------|-------------------|
| taskwarrior | タスク管理特化。時間ブロック概念なし |
| calcurse / calcure | カレンダー寄り。ブロック単位の管理ではない |
| girok | スケジューラだがzoom切替なし |

### GUI/Web
| Name | Diff from ytasky |
|------|-------------------|
| Sunsama | コンセプト近いがGUI/有料。TUI版が欲しい |
| TimeBloc | タイムブロック特化だがモバイルアプリ |
| Motion | AI自動スケジュール。手動で組むytaskyとは方向が異なる |

### 差別化ポイント
1. **TUI + Vim操作**: ターミナル完結。lazygit/yazi世代向け
2. **15分ブロック粒度**: 曖昧な「午前中」ではなく厳密な時間管理
3. **生活全体カバー**: 睡眠・食事も含めた24h管理で「余り」を可視化
4. **3段階ズーム**: 日↔週↔月をシームレスに行き来
